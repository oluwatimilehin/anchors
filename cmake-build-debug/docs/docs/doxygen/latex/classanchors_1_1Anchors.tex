\doxysubsection{Detailed Description}
\mbox{\hyperlink{classanchors_1_1Anchors}{Anchors}} is an utility class containing functions to simplify creating a shared pointer to an \mbox{\hyperlink{classanchors_1_1Anchor}{Anchor}}, which the \mbox{\hyperlink{classanchors_1_1Engine}{Engine}} class operates on. 

To create shared pointers with custom allocators, you can invoke the public constructors in the {\ttfamily \mbox{\hyperlink{classanchors_1_1Anchor}{Anchor}}} class.
