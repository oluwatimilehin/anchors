\doxysubsection{Detailed Description}
\mbox{\hyperlink{classanchors_1_1Engine}{Engine}} is the brain of Anchors, containing the necessary functions and data to retrieve the value of an {\ttfamily \mbox{\hyperlink{classanchors_1_1Anchor}{Anchor}}} object. 

Note that this class is not thread-\/safe.

Returns the value of the given \mbox{\hyperlink{classanchors_1_1Anchor}{Anchor}}. 

This function is only guaranteed to return the latest value of an \mbox{\hyperlink{classanchors_1_1Anchor}{Anchor}} marked observed using {\ttfamily \mbox{\hyperlink{classanchors_1_1Engine_a8b358a80ca9c9e9a44b3e219250ff783}{observe()}}}.
